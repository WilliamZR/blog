Question Answering

  • Two major paradigms:

    • Information retrieval(IR)-based (open domain)

      Information retrieval is used to find relevant passages

      Reading Comprehension to answer directly from spans of text

    • Knowledge-based

      build a semantic representation of the query

      Meaning representations are used to query databases of facts.

  • Factoid Questions: can be answered in simple facts in short texts

Information Retrieval

  • Information Retrieval: retrieval of all manner fo media based on user information needs

  • ad hoc retrieval

    • user poses a query

    • returns ordered set of documents from some collection

Term Weighting

  • We use term weight instead of raw counts (TF-IDF, BM25)

  • TF-IDF

    $$
    \mathrm{tf}_{t, d}=\log {10}(\operatorname{count}(t, d)+1)\
    \mathrm{idf}    {t}=\log {10} \frac{N}{\mathrm{df}{t}}
    $$

Inverted Index

  • Basic Search Problem: find all documents $d \in C$ containing a term $q \in Q$

  • Inverted index consists of two parts:

    • dictionary: a list of terms, each term pointing to a posting list

    • postings list: list of documents IDs associated with each term, which can alse contain term frequency information.

Entity Linking

  • Entity Linking: association a mention in text with the representation of some real-world entity in an ontology.

  • Two stages

    • Mention Detection

    • Mention Disambiguation

Linking Based on Anchor Dictionaries and Web Graph

  • Creation of Anchor Dictionary

    • Each page $e$ as entity

    • in-link($e$)

    • Anchor dictionary： lists of pages point to it (anchor texts, spans of text on other pages that point to it)

    • $freq(a)$ : frequency in Wikipedia, including non-anchor uses

    • $linkprob(a)$ : $\frac{link(a)}{freq(a)}$

    • Clean up: remove single characters, pure numbers, low linkprob strings.

  • Mention Detection

    querying token sequence with window up to 6 words

  • Mention Disambiguation

    • Many spans are ambiguous, match anchors for multiple wiki pages

    • Two factors for diaambiguation:

      • prior probability:

        $$
        p(e|a) = \frac{count(a\rightarrow e)}{link(a)}
        $$

      • relatedness

        the relatedness of this entity to other entities in the input question
